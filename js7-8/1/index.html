JavaScript

Автор языка - Брендан Эйх(Айх) - специалист компании Netscape, написал язык в 1995г. в возрасте 34 лет. С 2005г. по 2014г. Брендан Эйх занимал пост директора компании "Mozilla Corporation".

В 1995г. проведена стандартизация языка ассоциацией ECMA (Ecma International — ассоциация по стандартизации информационных и коммуникационных технологий).

Две очень важные возможности языка: замыкания и контекст выполнения.

Замыкания – это возможность обращаться к переменным, которые доступны в месте объявления функции. Замыкания — это функции, ссылающиеся на независимые (свободные) переменные. Другими словами, функция, определённая в замыкании, «запоминает» окружение, в котором она была создана.

Контекст выполнения – это изменения ссылки this на объекты внутри функции, в зависимости от того, для кого выполняется функция.

Cамая свежая информация по наиболее актуальным вопросам JavaScript на https://developer.mozilla.org/ru/docs/JavaScript.

jQuery

jQuery — javascript библиотека, использование которой делает разработку javascript кода намного быстрее, проще и приятнее. Методы jQuery можно связывать в цепочки.

В jQuery реализована возможность выполнения запросов к серверу без перезагрузки страницы, так называемая технология ajax.

Сейчас существует две основные ветки версий jQuery 1.x и 2.x. Их отличия заключаются лишь в том, что в версиях 2.x перестали поддерживаться браузеры IE версий 8, 7 и 6.

Определения

CMS - если говорить упрощенно, это программа, которая помогает владельцам управлять содержимым сайта, не требуя от них при этом каких-либо особых знаний и навыков. CMS (WordPress, Joomla, Drupal) выполняют две основные функции: формируют страницы для пользователей и помогают владельцам управлять содержимым сайта.

Angular JS - opensource JavaScript-фреймворк.

Полизаполнения (polyfills) – это решения на javaScript, имитирующие работу новых функций HTML5 в устаревших браузерах.
